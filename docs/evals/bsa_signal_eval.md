# BSA Signal Evaluation

## Purpose

Evaluate whether BSA output correctly separates background, structure, and
action, especially for true and false breakout decisions.

## Rubric

- Background supports or rejects the trade direction.
- Structure identifies key support, resistance, range, breakout, or pullback.
- Action gives valid entry, invalidation, and risk handling.
- False breakout risk is explicitly marked when follow-through is missing.

## Error Types

- `WRONG_STRUCTURE`
- `FALSE_BREAKOUT_MISS`
- `BAD_ACTION`
- `BAD_INVALIDATION`
- `OVERCONFIDENT`
