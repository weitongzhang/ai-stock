# Theme Strength Evaluation

## Purpose

Evaluate whether theme strength scoring correctly uses F-M-C-T evidence:
Flow, Map, Core, and Timing.

## Core Metrics

- Theme classification accuracy.
- Ranking consistency against human or delayed outcome labels.
- Evidence coverage across F/M/C/T dimensions.
- False-positive control for news-only themes.

## Error Types

- `WRONG_THEME`
- `NO_EVIDENCE`
- `OVERCONFIDENT`
