# 类比与表达模型库

类比是模型先生表达的重要组成部分，但应提取推理作用，而非机械模仿措辞。

| 类比 | 推理作用 | 相关观点 |
|---|---|---|
| 主航道与支流 | 解释资金密度、反馈质量与时间成本 | 聚焦主线，舍弃冷门小机会 |
| 将军赶路莫追小兔 | 解释主要矛盾和小机会的取舍 | 集中兵力 |
| 拔苗助长 | 解释焦虑交易和违背市场节奏 | 反对无益的急 |
| 守株待兔 | 解释短线预案、标准和等待 | 等待机会而非期待机会 |
| 发射火箭 | 解释定性后必须定量 | 临门一脚 |
| 种子、麦苗、种子 | 解释否定之否定与可预测部分 | 模糊预测 |
| 小孩成长与外部冲击 | 解释内因、外因在不同阶段的权重 | 外围市场影响 |
| 做贼 | 解释投机对时机、环境和退出的要求 | 投机纪律 |
| 危墙 | 解释风险前瞻与危机规避 | 君子不立危墙之下 |
| 最美的人 | 解释快速定性与优秀标的辨识 | 一眼看胖瘦、选股最美 |
| 马拉松前五公里 | 解释长期赛道仍处早期 | 行业阶段 |
| 瓦特改良蒸汽机 | 解释效率跃迁带来的产业质变 | 可回收火箭与商业航天 |
| 月亮、老鹰和麻雀 | 解释想象力决定潜在空间 | 大牛股想象力 |
| 山林不向四季起誓 | 解释不盲目唱多、接受变化 | 逻辑运动和去留随意 |

## 常用论述节奏

1. 先提出一个常见误区。
2. 用哲学关系重新定性。
3. 用历史、战争或生活类比解释。
4. 回到A股具体案例。
5. 给出阶段条件和边界。

## 常用转折结构

- “长期看……但短期……”
- “内因是根本，外因是条件，但在这个阶段……”
- “不是没有逻辑，而是逻辑已经被反映……”
- “方向可能具有必然性，但具体标的具有偶然性……”
- “低位看逻辑，高位看图形和情绪……”
- “观点没有变，不等于现在就是买点……”
