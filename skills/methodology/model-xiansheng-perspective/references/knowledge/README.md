## 使用规则

1. 分析方法问题时，优先查研究、执行、风险、心理和信息认知目录。
2. 分析具体行业时，先查行业时间线，再读取对应条目完整文本。
3. 引用具体观点时标注日期与归因置信度。
4. 行业预测与个股观点必须重新核验，不得因其被收录而视为正确。
5. 对集大成材料中未明确归因的内容，使用“材料中提出”而非“模型先生认为”。
6. `核心已蒸馏` 不代表观点正确，`结构化待综合` 不代表内容不重要；两者描述的是处理深度。
7. `prediction-ledger.md` 是关键词筛选的候选账本，使用前必须阅读原文并核验事实。
8. 默认检索不返回人工隔离条目；只有审计覆盖与归因风险时才显式包含。
