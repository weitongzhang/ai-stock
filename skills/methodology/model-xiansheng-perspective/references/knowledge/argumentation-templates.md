# 论证模板库

以下是语料中反复出现的推理结构。它们比具体结论更值得复用。

## 关系总和模板

1. 列出影响对象的全部主要关系：业绩、题材、资金、指数、政策、行业。
2. 判断当前阶段哪一条关系成为主要矛盾。
3. 说明关系何时会转化。
4. 给出主导关系改变后的证伪条件。

适用：个股走势、行业阶段、消息影响。参考：`MX-0012`、`MX-0181`。

## 质变识别模板

1. 说明此前的量变积累。
2. 找到公司、行业、技术、组织或重组发生跃迁的证据。
3. 区分真实质变与概念故事。
4. 判断市场是否已经充分计价。
5. 跟踪质变是否继续强化。

适用：成长股、困境反转、新产业。参考：`MX-0020`、`MX-0138`。

## 主线与主要矛盾模板

1. 顾得全：对板块分类排队。
2. 看得准：观察资金真正流向。
3. 确认龙头和中军的市场地位。
4. 识别主次矛盾切换和新旧龙头替换。
5. 舍弃不能决定全局的小机会。

适用：复盘、主线识别、龙头判断。参考：`MX-0013`、`MX-0022`、`MX-0221`、`MX-0222`。

## 必然与偶然模板

1. 找到可重复、可解释的大趋势或规律。
2. 标出具体载体、时间和路径中的偶然性。
3. 不把行业必然性直接等同于个股必然上涨。
4. 用市场选择和证伪条件处理偶然部分。

适用：产业趋势、牛股选择、市场预测。参考：`MX-0023`、`MX-0058`、`MX-0109`。

## 内因与外因模板

1. 判断对象自身结构和趋势是否健康。
2. 判断外部事件在当前阶段是条件还是主导因素。
3. 评估对象对外因的敏感程度。
4. 外因消退后重新检查内因。

适用：外围暴跌、政策冲击、突发事件。参考：`MX-0031`。

## 位置、预期与消息模板

1. 消息是否已经被价格提前反映。
2. 好消息落地后是否失去新增驱动。
3. 坏消息是否已在下跌中消化。
4. 当前是炒预期、兑现还是证伪阶段。

适用：财报、事件驱动、利好利空。参考：`MX-0027`、`MX-0112`。

## 从普遍性到特殊性模板

1. 找市场相对强的普遍方向。
2. 在方向内寻找更强的细分关系。
3. 在细分内部寻找具有独特性的标的。
4. 持续深入，直到特殊性被市场选择验证。

适用：细分赛道、龙头、中军、侧翼战。参考：`MX-0054`、`MX-0091`、`MX-0193`。

## 两点论风险模板

1. 写出向上成立条件。
2. 写出向下和最坏结果。
3. 明确错误发生时的止损、减仓或退出方式。
4. 判断最坏结果是否可承受。

适用：任何买卖与持有判断。参考：`MX-0024`、`MX-0199`。

## 历史感复盘模板

1. 只使用当时可以获得的信息。
2. 还原当时市场情绪与决策压力。
3. 区分决策质量和最终结果。
4. 提炼可迁移规律，不复制历史表面形态。

适用：牛股复盘、失败复盘、历史类比。参考：`MX-0018`、`MX-0030`、`MX-0116`。

## 逻辑运动与纠错模板

1. 记录原始逻辑和预期。
2. 跟踪事实是否强化、弱化或证伪。
3. 区分价格波动与逻辑变化。
4. 逻辑不在时，不因成本和过去观点继续维护。

适用：持仓跟踪、卖点、观点更新。参考：`MX-0108`、`MX-0219`、`MX-0220`。
