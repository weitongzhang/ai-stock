# 案例模式库

具体股票与行业观点会过期，但案例背后的结构可以复用。

## 从0到1与从1到10

- 结构：新技术或新产业完成可用性突破，估值体系随发展阶段变化。
- 观察：技术验证、量产、需求、利润、产业链位置。
- 风险：把实验性进展误判为商业质变。
- 案例入口：固态电池、赛力斯、AI应用相关条目。

## 自主科技与外部产业链

- 结构：自主可控逻辑与海外龙头映射逻辑具有不同主要矛盾。
- 观察：国产替代、稀缺性、产业地位、资本市场代表性。
- 风险：只看题材标签，不看真实产业能力。
- 案例入口：寒武纪、科创芯片、光模块相关条目。

## 周期资源品

- 结构：商品供需周期、全球定价、资本开支与股票表现共同作用。
- 观察：供需缺口、库存、定价权、商品与股票的关系。
- 风险：长期逻辑正确但买点、持有周期和波动承受能力不匹配。
- 案例入口：铜、铝、黄金、有色相关条目。

## 宏大叙事题材

- 结构：时代呼声、产业空间、政策与资金共同形成长期想象力。
- 观察：从概念炒作走向基础设施、商业化和业绩的路径。
- 风险：宏大空间无法落实到具体公司。
- 案例入口：商业航天、AI应用、机器人相关条目。

## 正宗龙头到侧翼分支

- 结构：第一阶段由正宗龙头形成辨识度；共识和拥挤加深后，资金转向大方向内的侧翼。
- 观察：资金阻力、量化参与、细分关系、后期确认。
- 风险：把任何杂毛解释为侧翼。
- 案例入口：侧翼战、谷歌产业链、机器人零部件相关条目。

## 预期兑现博弈

- 结构：价格提前交易业绩或消息，落地后新增驱动消失。
- 观察：预期形成时间、涨幅、实际结果与预期差。
- 风险：把所有利好兑现都机械视为卖点。
- 案例入口：业绩博弈、好消息和坏消息相关条目。

## 超跌与困境反转

- 结构：第一次超跌反弹来自情绪和估值修复；后续上涨需要基本面改善。
- 观察：跌幅、反弹阶段、行业改善、质变证据。
- 风险：反复用超跌理由持有长期衰退对象。
- 案例入口：白马、超跌股、光伏相关条目。

## 龙头、中军与地位股

- 结构：龙头提供方向与辨识度，中军提供容量和板块稳定性，角色会随阶段轮动。
- 观察：行业地位、资金容量、强度、市场代表性。
- 风险：把历史地位当永久地位。
- 案例入口：科创芯片中军、创新药中军、龙头逻辑相关条目。

## 高位图形与情绪

- 结构：低位由逻辑和预期差主导，高位由资金、图形和情绪主导。
- 观察：异动、加速、宽幅震荡、破位和参与者变化。
- 风险：用长期基本面否认高位结构恶化。
- 案例入口：严重异动、低位看逻辑高位看图形相关条目。
