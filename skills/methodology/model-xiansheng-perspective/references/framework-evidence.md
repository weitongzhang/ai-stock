# 框架证据与反证

## 语料说明

本次使用两份用户提供材料：

1. 《模型先生语录合集（全20260208）》：覆盖约2023年12月至2025年12月的大量视频语录。
2. 《模型先生包括评论区有价值的-集大成之作》：覆盖2026年观点、评论区回复和整理者分析。

两份文件约35万字符。第二份明确混合了博主回复与整理者观点，不能全部视为模型先生原话。

## 三重验证

### 辩证法与市场具体实践

- **跨域复现**：用于市场结构、个股定价、外部冲击、交易心理和社会历史。
- **生成力**：面对新问题时会先找主要矛盾、内外因与阶段变化。
- **排他性**：明确称自己的逻辑是“辩证法与A股的具体实践相结合”。

### 质变公司

- **跨域复现**：用于公司经营、行业变化、技术突破、重组、社会与历史。
- **生成力**：面对新公司会追问“是否发生非一次性的质变”。
- **排他性**：反复把“质变”作为其投资体系的核心术语。

### 必然与偶然

- **跨域复现**：用于趋势预测、龙头选择、个人成功和风险管理。
- **生成力**：会押注大方向必然性，同时用纪律管理具体路径偶然性。
- **排他性**：明确反对把投资基石放在偶然与侥幸上。

### 主观服从客观

- **跨域复现**：用于龙头选择、基本面与走势冲突、预测纠错。
- **生成力**：当市场选择与原偏好冲突时，优先重新审视客观事实。
- **排他性**：提出“可以不尊重资金的智慧，但必须尊重资金的力量”。

### 主要矛盾与主航道

- **跨域复现**：用于板块排序、持仓集中、等待机会、市场风格切换。
- **生成力**：面对复杂市场会先排序并舍弃次要机会。
- **排他性**：大量使用战争和主航道类比组织交易决策。

### 分阶段切换工具

- **跨域复现**：用于周期股、成长股、题材股、指数与高位龙头。
- **生成力**：会根据位置判断当前应该看逻辑、业绩、预期、图形还是情绪。
- **排他性**：“低位看逻辑，高位看图形和情绪”是高辨识度规则。

## 遗漏审计补充

对351个正文条目建立主题索引后，发现第一版对以下框架覆盖不足：

| 框架 | 语料中的作用 | 修订后的调用位置 |
|---|---|---|
| 定性、定量与临门一脚 | 把性质判断转化为数据、空间与买卖条件 | 研究工作流、输出模板 |
| 系统四项质量 | 检查完整性、结构性、层次性、开放进化性 | 关键二级框架 |
| 两点论 | 同时准备上涨与下跌，承认概率而非百分百 | 风险与反证 |
| 整体、过程与历史感 | 防碎片化、结果偏见和静止分析 | 研究工作流 |
| 特殊性与侧翼战 | 从大方向逐层寻找独特关系和后期分支 | 选股与市场结构 |
| 三张底牌 | 区分价值、规律与违规漏洞 | 机会来源与诚实边界 |
| 想象力立足现实 | 评估大牛股远期空间，同时要求现实证据 | 质变与估值 |
| 计划与客观灵活 | 兼顾计划性和市场运动变化 | 执行与纠错 |

行业观点数量很大，但具有强时效性。自主科技、有色、创新药、AI应用、商业航天等只作为带日期的案例，不能固化为Skill结论。

## 主要反证与风险

1. 语料中存在极明确的指数、时间与市值预测，与“模糊正确”原则存在张力。
2. 成功案例和事后解释较多，缺少完整失败记录与统计检验。
3. “质变”“主要矛盾”“市场选择”定义灵活，可能被用于事后合理化。
4. 集中仓位和“心狠手狠”若脱离证据、止损和风险承受力，危险性很高。
5. 长期观点与视频发布时间并非买点，用户很容易误读。

## 使用约束

- 所有具体公司分析都必须核验最新事实。
- 必须同时列支持证据、反方证据和证伪条件。
- 不把历史预测命中视为未来预测能力证明。
- 不采用语料中的明确收益或市值目标作为输出结论。
