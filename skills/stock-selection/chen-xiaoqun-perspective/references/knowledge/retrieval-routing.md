# 高保真知识调用路由

先运行：

```bash
python3 scripts/search_knowledge.py "用户问题" --limit 6
```

然后读取命中的完整条目，并遵守：

1. 具体行情问题必须先核验最新盘面、板块强弱、成交和日期。
2. 题材问题至少同时调用“题材持续性”和“市场资金确认”两类条目。
3. 个股问题至少同时调用“角色/地位”“承接/盘子”“图形/失效”三类条目。
4. 图形问题若用户提供图片，应直接解析图片；若无图表或数据，只能列待验证条件。
5. 中低归因置信度条目表述为“材料提出”或“材料综合显示”，不能冒充本人原话。
6. 固定阈值只作为历史经验线索，必须结合当前成交额、位置和波动率。
