# 陈小群交易视角核心观点

1. **先题材，后个股**：主线来自持续发酵的共同注意力，不来自单日热榜。
2. **龙头由市场选出**：辨识度、带动力、题材土壤、换手承接和盘子适配共同决定。
3. **分歧不等于结束**：健康分歧完成筹码交换；无承接的分歧才意味着结构弱化。
4. **图形是成绩单**：月周日与分时分别观察空间、节奏、结构与执行，不能迷信孤立K线。
5. **规律只提供概率**：寻找多数案例的普遍性，同时保留例外、失败和环境变化。
6. **阶段决定工具**：发酵期看持续性，主升期看核心与承接，高位看结构与退出，主跌期停止用故事解释价格。
