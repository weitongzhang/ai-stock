# 舆情监控风险分层

## 一级风险：red

这类事件会直接降低或否决技术买点权重。即使 K 线出现反弹，也先按风险释放或超跌反弹处理。

- 监管立案、证监会立案、交易所纪律处分、行政处罚、重大违法
- 财务造假、虚假记载、误导性陈述、信披违法违规
- 退市风险、ST、暂停上市、审计无法表示意见
- 债务违约、流动性危机、银行账户冻结、破产重整
- 大额诉讼、仲裁、资产查封、实控人被留置或调查
- 巨额亏损、净资产为负、持续经营重大不确定性

交易影响：

```text
技术买点降权
反弹先看减仓或观察
必须设置硬止损
不使用亏损摊平逻辑
```

## 二级风险：orange

这类事件不一定否决交易，但会提高不确定性，要求更强的价格确认。

- 异动公告、澄清公告、媒体报道澄清
- 问询函、关注函、监管工作函
- 业绩预亏、业绩大幅下滑、商誉减值
- 股东减持、限售解禁、质押比例较高
- 股价短期大幅波动、题材过热、龙虎榜分歧
- 重大事项停牌、并购重组不确定

交易影响：

```text
突破必须有 follow-through
仓位低于普通技术信号
优先等待回踩确认
短线目标更保守
```

## 正向事件：green

正向事件只提供观察理由，不等于直接买点。必须等待价格确认。

- 重大合同、中标、订单、海外项目
- 业绩预增、扭亏、毛利率改善
- 回购、增持、股权激励
- 新产品、新技术、产业政策支持
- 行业景气改善、涨价、产能释放

交易影响：

```text
若价格未确认，加入观察池
若价格突破并有跟随，可提高候选权重
若高位放量冲高，警惕利好兑现
```

## 与价格行为方法论的结合

```text
舆情决定风险底色
价格行为决定市场是否确认
成交量确认急迫感
后续跟随决定真假
风险标签决定仓位和止损严格度
```

典型规则：

- red + 弱势 K 线：回避或减仓。
- red + 强反弹：先当作事件风险释放后的超跌反弹，不当作趋势恢复。
- orange + 突破：需要比普通突破更强的后续跟随。
- green + 无突破：只观察，不追。
- green + 高位放量无跟随：警惕利好兑现或诱多。
