# LIFT 板块龙头识别框架

全称：Lead-Impact-Flow-Trend Sector Leader Framework

中文名：领先-影响-资金-趋势 龙头识别框架

研究状态：外部方法论沉淀稿。规则未经系统样本验证、回测和长期复盘前，仅用于技术研究、选股训练和复盘框架，不构成投资建议。

## 1. 方法论定义

LIFT 是一套用于识别 A股板块龙头的框架。它解决的问题不是“哪只股票今天涨得最多”，而是：

```text
在一个有效板块行情中，哪只股票最先启动、最能带动情绪、最被资金反复攻击、趋势结构最健康。
```

LIFT 分为四层：

```text
L = Lead 领先性
I = Impact 带动性
F = Flow 资金强度
T = Trend 趋势结构
```

一句话定义：

```text
板块龙头是板块行情中最早体现方向、最强吸引资金、最能带动扩散、回调最抗跌并能持续获得后续跟随的核心标的。
```

## 2. 先确认板块有效

找龙头之前，必须先确认板块本身是否有效。没有有效板块，就不要急着给个股贴“龙头”标签。

有效板块通常具备：

- 板块指数强于大盘。
- 板块指数放量突破或脱离底部区间。
- 板块内涨停、大涨、创新高个股增加。
- 板块行情至少有 2 到 3 天后续跟随。
- 有明确催化，如政策、产业趋势、订单、业绩、事件或资金风格切换。
- 板块内出现明显扩散，从少数个股走强到多只个股跟随。

板块无效或仅一日脉冲的表现：

- 只有一只个股独涨，板块不跟。
- 当天冲高后没有第二天跟随。
- 板块指数没有放量，没有突破关键位置。
- 新闻催化模糊，资金只做套利。
- 龙头候选高开低走，跟随股快速熄火。

规则：

```text
先判板块，再选龙头。
板块无效时，龙头识别降级为个股短线交易，不做板块主线判断。
```

## 3. L：Lead 领先性

领先性回答：

```text
谁最早告诉市场这个板块有机会？
```

观察维度：

- 最早启动。
- 最早突破平台。
- 最早创阶段新高。
- 大盘弱时率先走强。
- 板块回调后率先修复。
- 板块二次启动时率先表态。
- 在同板块中最早形成强趋势结构。

高质量领先信号：

- 板块还未全面扩散时，个股已经放量突破。
- 大盘或板块调整时，个股拒绝下跌或快速收回。
- 板块再次走强时，它先于其他个股突破前高。

需要避免：

- 把后排补涨误判为领先龙头。
- 把高开套利误判为启动。
- 只按当天涨幅排序，不看启动顺序。

## 4. I：Impact 带动性

带动性回答：

```text
它涨的时候，板块会不会跟；它弱的时候，板块会不会降温？
```

观察维度：

- 它拉升后，同板块个股跟涨。
- 它涨停或大涨后，板块内涨停数量增加。
- 它炸板或回落后，板块情绪明显降温。
- 市场讨论度、辨识度高。
- 与板块核心逻辑最正宗、最贴合。
- 能吸引跨板块资金关注。

带动性强的表现：

- 个股先动，板块指数随后上行。
- 个股封板后，后排出现批量跟风。
- 个股回封时，板块情绪恢复。
- 它的分时强弱影响同板块交易者决策。

带动性弱的表现：

- 自己涨，但板块不跟。
- 板块涨，它只是被动跟随。
- 涨停后没有扩散。
- 回落后板块无感。

## 5. F：Flow 资金强度

资金强度回答：

```text
资金是否愿意持续在它身上交易？
```

观察维度：

- 成交额是否显著放大。
- 换手是否充分。
- 放量上涨、缩量回调是否成立。
- 封板质量、炸板后的回封能力。
- 大单/主力资金是否反复攻击。
- 龙虎榜、机构、游资、北向或量化资金是否参与。
- 成交额容量是否支持大资金进出。

资金强但健康：

- 突破时放量。
- 回调时缩量。
- 高位换手后仍能继续走强。
- 炸板后能快速回封或次日修复。
- 成交额大但价格没有明显失控。

资金强但危险：

- 高位爆量长上影。
- 连续加速后无换手。
- 成交额急剧放大但收盘弱。
- 板块跟随不够，只有个股抱团。

规则：

```text
龙头需要资金强度，但资金强度必须和趋势结构、板块带动一起判断。
```

## 6. T：Trend 趋势结构

趋势结构回答：

```text
它的价格结构是否支持继续成为龙头？
```

观察维度：

- 是否在 MA5/MA10/MA20 上方运行。
- 是否突破平台、箱体、前高或历史压力。
- 是否强于板块指数。
- 是否强于同板块多数个股。
- 回调是否不破关键支撑。
- 突破后是否有后续跟随。
- 是否形成更高高点和更高低点。

高质量趋势结构：

- 平台突破后连续站稳。
- 回踩不破突破位。
- 调整时缩量，重新上攻时放量。
- 板块调整时它最抗跌。
- 板块修复时它最先创新高。

结构失效：

- 跌破关键突破位后无法收回。
- 放量跌破 MA10 或核心支撑。
- 高位连续长上影，无后续跟随。
- 补涨股接力，它自己不再创新高。

## 7. 龙头类型

不要强行只选一个龙头。A股行情中经常有多层龙头。

```text
总龙头：全市场情绪核心，跨板块吸引资金。
板块龙头：该板块最强、最有带动性的核心标的。
趋势龙头：不一定连板，但趋势最稳、最适合跟踪。
弹性龙头：涨幅最快，适合短线情绪，但波动大。
容量龙头：成交额大，大资金可进出，适合机构/大资金偏好。
补涨龙头：主龙头高位后，低位接力扩散，地位低于主龙头。
反包龙头：分歧后率先修复，体现资金承接。
防御龙头：板块退潮时最抗跌，可能成为下一轮修复核心。
```

排序原则：

```text
总龙头 > 板块龙头 > 趋势龙头/容量龙头 > 弹性龙头 > 补涨龙头。
```

## 8. LIFT 评分表

总分 100 分。

```text
板块有效性：0-20
领先性 Lead：0-15
带动性 Impact：0-15
资金强度 Flow：0-15
趋势结构 Trend：0-15
逻辑正宗度：0-10
风险位置：0-10
```

评分解释：

```text
80-100：核心龙头候选，可重点跟踪。
65-79：强势跟随股或阶段龙头，需要 BSA 二次确认。
50-64：普通板块个股，只做观察或低级别交易。
0-49：不参与龙头筛选。
```

## 9. 龙头确认流程

```text
1. 找到当日或近几日最强板块。
2. 确认板块不是一日脉冲，有无后续跟随。
3. 列出板块内涨幅、成交额、创新高、涨停、强趋势个股。
4. 按启动顺序排序，找最早表态者。
5. 观察谁拉升时板块跟随，谁回落时板块降温。
6. 对候选股做 LIFT 评分。
7. 对 80 分以上候选，用 BSA 判断交易位置。
8. 每日复盘龙头地位是否转移。
```

## 10. 龙头地位转移

龙头不是永久称号。需要每天检查地位是否转移。

可能发生转移的信号：

- 原龙头高位放量滞涨。
- 原龙头跌破关键支撑，无法修复。
- 新个股率先突破并带动板块。
- 补涨股变成新的情绪核心。
- 板块逻辑从一个分支切到另一个分支。
- 容量资金从弹性小票切到大市值核心。

判断规则：

```text
谁先修复，谁有机会接棒。
谁能带动扩散，谁才可能成为新龙头。
只有涨幅，没有带动，不算地位转移。
```

## 11. 与 BSA 的关系

LIFT 和 BSA 分工明确：

```text
LIFT 解决：选谁是板块核心。
BSA 解决：什么时候交易、在哪里止损、如何处理无跟随。
```

组合流程：

```text
板块有效性确认
  -> LIFT 龙头识别
  -> BSA 背景/结构/行动确认
  -> 交易计划
  -> 复盘龙头地位和交易质量
```

LIFT 高分但 BSA 位置差时，不追。

BSA 位置好但 LIFT 地位弱时，降低预期，只当普通交易。

## 12. 与 YC-buy 的关系

YC-buy 可以提供技术买点候选，LIFT 用于判断这些候选是否属于板块核心。

加分：

- YC-buy 触发买点，同时该股是 LIFT 高分龙头候选。
- 买点发生在有效板块行情中。
- 买点方向与板块主线一致。
- 买点后有板块跟随。

降级：

- YC-buy 有买点，但板块无效。
- 个股是后排补涨，不具备带动性。
- 买点发生在高位情绪末端。
- 龙头已经切换，原标的只剩反抽。

## 13. 最小可执行版本

```text
1. 先确认板块有效。
2. 找最早启动、最早突破、最先修复的个股。
3. 看它是否能带动同板块扩散。
4. 看成交额、换手和放量缩量是否健康。
5. 看趋势是否强于板块和大盘。
6. 打 LIFT 分数，80 分以上才列为核心龙头候选。
7. 用 BSA 决定是否能交易，不能因为是龙头就无脑追。
```
