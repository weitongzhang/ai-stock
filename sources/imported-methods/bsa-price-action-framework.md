# BSA 价格行为框架

全称：Background-Structure-Action Price Action Framework

中文名：背景-结构-行动价格行为框架

来源：融合方方土价格行为方法论与 Al Brooks 价格行为交易方法论后沉淀形成。

研究状态：外部方法论融合稿。规则未经系统样本验证、回测和长期复盘前，仅用于技术研究、交易训练和复盘框架，不构成投资建议。

## 1. 命名含义

BSA 分为三层：

```text
B = Background 背景
S = Structure 结构
A = Action 行动
```

它的核心思想是：

```text
先判断市场背景，再识别价格结构，最后制定可执行行动。
```

更完整的执行链条是：

```text
背景 -> 结构 -> 急迫感 -> 信号 -> 跟随 -> 风险 -> 行动 -> 复盘
```

## 2. 两套方法论的分工

### 方方土方法论负责

- 判断市场背景。
- 强调关键位置。
- 用成交量确认或质疑急迫感。
- 用后续跟随验证真假突破、真假反转。
- 强调风险预算和交易反馈。

### Al Brooks 方法论负责

- 细分市场状态。
- 提供价格行为结构语言。
- 定义 H1/H2、L1/L2、Always In、二次入场、突破回调、失败突破、主要趋势反转。
- 提供更细的入场、止损、目标和无跟随退出规则。
- 支持逐K训练和结构化复盘。

## 3. 融合后的主流程

```text
1. 方方土定背景：趋势、区间、突破、高潮、反转尝试。
2. Brooks 细分状态：强趋势、通道、交易区间、突破阶段、主要趋势反转。
3. Brooks 定结构：H2/L2、突破回调、失败突破、二次入场、楔形、Always In。
4. 方方土看急迫感：成交量、强收盘、是否远离关键位。
5. 双方共同确认：信号后是否有 1 到 3 根后续跟随。
6. Brooks 定执行：入场、止损、目标、scalp/swing、无跟随退出。
7. 方方土定反馈：持有、减仓、降低目标、退出。
8. 复盘归因：背景错、结构错、急迫感错、执行错，还是风险控制错。
```

## 4. BSA 的一句话定义

```text
BSA 是一套以市场背景为前提、以价格结构为语言、以量价急迫感和后续跟随为确认、以风险计划为行动边界的价格行为交易框架。
```

## 7. BSA 禁止清单

- 不在背景不清楚时强行交易。
- 不在区间中间交易。
- 不追没有跟随的突破。
- 不把成交量当成独立买卖信号。
- 不把 Brooks 形态当成脱离背景的固定公式。
- 不把第一次逆势信号当主要趋势反转。
- 不做止损无法定义的交易。
- 不让单笔风险超过预设预算。

## 8. 真假突破判定模块

真假突破属于 BSA 的 S+A 交界模块：先在 S 层确认突破的是关键结构位，再在 A 层观察是否被市场接受。

核心定义：

```text
真突破 = 突破关键位后，价格被市场接受，并出现后续跟随。
假突破 = 突破关键位后，价格被市场拒绝，并重新跌回原结构。
```

### 8.1 真突破条件

真突破至少需要满足以下多数条件：

- B 背景支持当前方向，最好处于强板块、强指数环境或个股主线中。
- S 结构突破的是前高、平台上沿、趋势线、均线密集区、箱体边界等关键位。
- 突破K线实体较强，收盘接近高位，而不是长上影。
- 成交量放大，但不是放量滞涨。
- 突破后 1 到 3 根K线不快速跌回关键位。
- 回踩关键位时缩量，且能守住。
- 再次上攻时有跟随买盘，能继续创新高或维持高位横盘。

执行含义：

```text
突破当下只能小仓试。
回踩关键位不破，才是更稳健的确认买点。
跌回关键位下方，突破失败，必须降低预期或退出。
```

### 8.2 假突破条件

假突破通常出现以下特征：

- 背景不支持，指数弱、板块弱，或个股处在下降趋势反抽中。
- 突破发生在连续急拉后，缺少充分蓄势。
- 突破后出现长上影、冲高回落、放量滞涨。
- 收盘跌回关键位下方。
- 次日或后续 1 到 3 根K线不能重新站回关键位。
- 回踩没有承接，反而跌破突破K线低点。
- 后续没有跟随，成交量放大但价格不再推进。

执行含义：

```text
假突破不是补仓理由，而是风险信号。
跌回关键位下方，先退出或减仓。
如果重新站回关键位并出现跟随，再按新信号评估。
```

### 8.4 对胜率的作用

真假突破模块不能保证每次判断正确，但可以提高选择质量：

- 过滤弱背景里的冲高诱多。
- 避免在连续急拉后的末端突破追高。
- 把买点从“突破瞬间”后移到“回踩接受”。
- 用后续跟随确认资金是否愿意继续进攻。
- 让止损位置变得清楚，减少亏损扩大。

因此，BSA 中的真假突破判断主要提高的是两类胜率：一是选出更可靠的突破机会，二是降低假突破后的亏损幅度。

## 11. 与 LIFT 框架的关系

LIFT 板块龙头识别框架负责解决“选谁”，BSA 负责解决“怎么交易”。

```text
LIFT = 板块有效性 + 龙头地位识别
BSA = 背景 + 结构 + 行动计划
```

组合使用：

```text
先用 LIFT 找出板块核心标的
再用 BSA 判断是否处在可交易位置
```
