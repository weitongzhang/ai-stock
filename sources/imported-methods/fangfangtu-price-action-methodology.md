# 方方土价格行为方法论

来源背景：根据用户提供的方方土 / Louie PriceAction YouTube 视频链接，以及可公开访问的视频标题、摘要和镜像信息整理。本文件是方法论提炼，不是逐字稿。

研究状态：外部方法论沉淀。规则未经系统回测和样本验证前，先保存在 `sources/imported-methods/`；后续验证成熟后，再转化为明确的选股、回测、观察池或交易复盘逻辑。

风险说明：本文仅用于技术分析研究，不构成投资建议。

## 核心结论

方方土的方法不是简单的“量价公式”，而是一套以市场背景为优先级的价格行为框架。

```text
市场背景决定交易偏向。
市场结构决定交易位置是否有价值。
信号 K 线决定是否出现触发点。
后续跟随决定信号是真是假。
风险决定仓位大小。
市场反馈决定持有、减仓还是退出。
```

压缩成一条主线：

```text
背景 -> 结构 -> 信号 -> 跟随 -> 风险 -> 反馈
```

最实用的一句话：

```text
在正确的市场背景和关键位置，用价格行为识别买卖压力，用成交量确认急迫感，用后续跟随验证真假，用止损和仓位管理把概率优势长期执行出来。
```

## 不可丢失的核心逻辑

### 1. 背景优先，而不是信号优先

不要一上来就问“这根 K 线能不能买”。先判断当前市场环境：

- 趋势
- 交易区间 / 震荡
- 突破阶段
- 回调
- 高潮
- 反转尝试

同一根 K 线，在不同背景下意义完全不同。比如一根放量大阳线，可能是趋势启动，也可能是趋势延续，也可能是震荡区间上沿的诱多，还可能是趋势末端的高潮衰竭。

规则：

```text
先定义市场背景，再解释 K 线和成交量。
```

### 2. 价格行为是主线，成交量是证据

成交量不是独立买卖信号。成交量的作用，是确认或质疑价格行为里体现出来的急迫感。

正确逻辑：

```text
价格结构给出方向背景
K 线强弱显示短期买卖压力
成交量确认市场参与者是否急迫
后续走势验证判断是否正确
```

避免：

```text
放量 = 买
缩量 = 卖
```

应该使用：

```text
关键位置 + 强价格行为 + 合理成交量/急迫感 + 后续跟随 = 可交易优势
```

### 3. 关键位置比漂亮 K 线更重要

信号应该出现在其他交易者也会关注的位置：

- 趋势线
- 支撑和压力
- 交易区间上沿 / 下沿
- 前高 / 前低
- 突破点
- 回调区域
- 均线附近
- 测量目标位附近

如果位置本身不重要，再漂亮的信号 K 线也可能只是随机波动。

### 4. 真突破必须有后续跟随

突破不是价格穿过一条线就成立。真突破需要后续证据：

- 强突破 K 线
- 收盘靠近最高点 / 最低点
- 价格远离原区间
- 成交量或急迫感增强
- 后面继续出现同方向 K 线

关键点：

```text
第一根突破 K 线只是创造可能性
第二、第三根跟随 K 线提高确认度
没有后续跟随，突破可能是陷阱
```

### 5. 在交易区间里，先假设突破会失败

在交易区间 / 震荡里，市场经常制造诱多和诱空。默认行为不是追每一次突破。

区间默认逻辑：

```text
低买高卖
目标保守
早期突破先怀疑
只有强突破 + 后续跟随，才切换为趋势思维
```

这一条很重要，否则容易把方方土的方法误解成单纯的“追突破系统”。

### 6. 强运动后常常会尝试第二段

强突破、强反转或强趋势腿之后，市场经常尝试走第二段：

```text
第一段运动 -> 回调/测试 -> 第二段运动
```

观察重点：

- 回调是否浅？
- 逆势压力是否弱？
- 第二次推进是否出现？
- 第二段是否到达测量目标，还是提前失败？

第二段逻辑会影响入场、加仓、止盈和退出。

### 7. 一高 / 二高 / 三高 与 一低 / 二低 / 三低

方方土 / Al Brooks 价格行为体系里，经常会对趋势中的回调尝试进行计数：

```text
一高 / 二高 / 三高 = H1 / H2 / H3
一低 / 二低 / 三低 = L1 / L2 / L3
```

它们不是孤立地数“第一个高点、第二个高点、第三个高点”，而是趋势回调后，顺势恢复的第几次信号或触发尝试。

在多头趋势中：

```text
H1 = 回调后的第一次买入信号
H2 = 更深回调或两段回调后的第二次买入信号
H3 = 第三次买入信号，通常更晚，除非背景仍然很强，否则要更谨慎
```

在空头趋势中：

```text
L1 = 回调后的第一次卖出信号
L2 = 更深回调或两段回调后的第二次卖出信号
L3 = 第三次卖出信号，通常更晚，除非背景仍然很强，否则要更谨慎
```

核心思想：

```text
趋势 -> 回调 -> 第一次恢复趋势尝试 -> 如果失败或偏弱，等待第二次尝试
```

H2 / L2 往往比 H1 / L1 更重要，因为它代表市场经过回调测试后的第二次顺势尝试，经常和“第二段运动”连接在一起：

```text
第一段趋势腿 -> 两小段回调 -> H2/L2 信号 -> 第二段趋势腿尝试
```

实战解释：

- 强多头趋势里，H1 可能就足够，因为强趋势经常快速恢复。
- 弱趋势或交易区间里，H1 更容易失败。
- 两段回调后的 H2，通常是质量更好的趋势延续信号。
- H3 在极强趋势里也可能有效，但也可能意味着行情已经偏晚、拥挤或接近衰竭。
- 空头趋势里的 L1 / L2 / L3 是镜像逻辑。

不要机械计数。背景仍然第一：

```text
强趋势 + 浅回调 + 好信号 K = H1 可能足够
普通趋势 + 两段回调 + 好信号 K = H2 更值得关注
交易区间 + 区间上沿的 H1/H2 = 谨慎，区间逻辑优先
趋势末端 + 接近测量目标的 H3 = 可能有追高/衰竭风险
```

计数规则是描述市场行为的工具，不是自动入场指令。

### 8. 用测量目标判断空间

有效突破后，不是简单地“看多”或“看空”，还要估算潜在空间。

常见测量参考：

- 交易区间高度
- 前一段趋势腿幅度
- 突破前整理区间宽度

常见投射：

```text
目标位 ~= 突破点 +/- 前一区间高度
```

测量目标用于判断：

- 现在追入是否还有足够盈亏比
- 哪里适合部分止盈
- 价格是否已经接近衰竭或获利了结区域

### 9. 止损是交易逻辑失效点

止损不应该是随便设几个点。止损应该放在“如果价格走到这里，说明我的交易逻辑错了”的位置。

```text
如果价格到达这个位置，我对当前结构和信号的判断就失效。
```

只有先确定失效点，才能计算仓位。

正确顺序：

```text
交易机会 -> 失效点/止损 -> 止损距离 -> 仓位大小 -> 目标位
```

不要因为价格快打到止损，就临时把止损移远。

### 10. 仓位管理是心理管理的根源

小仓位不只是为了少亏钱，也是为了保护执行质量。

```text
仓位过大 -> 恐惧、过早止盈、不愿止损、报复交易
仓位合适 -> 看盘更清楚、执行更稳定、纪律更容易保持
```

“I don't care size”的实质是：

```text
仓位要小到一次正常亏损不会破坏判断力。
```

仓位公式：

```text
单笔最大亏损 = 账户资金 * 风险比例
仓位 = 单笔最大亏损 / 止损距离
```

保守研究默认值：

```text
单笔风险：0.5% 到 1.0%
激进主观交易上限：2.0%
```

### 11. 入场后根据市场反馈管理交易

入场后不能死拿，也不能随机退出。要看市场是否确认你的交易逻辑：

- 价格是否很快朝预期方向运动？
- 是否有后续跟随？
- 回调是否正常，还是过深？
- 是否出现强反向 K 线？
- 是否接近测量目标？
- 原来的市场背景是否已经改变？

交易管理：

```text
预期运动 + 后续跟随 -> 持有或移动止损
弱跟随/无跟随 -> 减仓、平推或降低目标
强反向 K 线 -> 退出或重新评估
到达测量目标 -> 止盈或收紧止损
```

## 常见交易模板

### 趋势回调

背景：

- 趋势已经建立。
- 回调到合理位置。
- 逆势 K 线开始变弱或重叠。

触发：

- 顺趋势方向出现信号 K。
- 成交量 / 急迫感支持趋势恢复。
- 入场后出现后续跟随。

风险：

- 止损放在回调失效点之外。
- 仓位由止损距离倒推。

### 突破延续

背景：

- 价格突破清晰的区间、整理、前高/前低或形态。

触发：

- 强突破 K 线。
- 收盘质量好。
- 成交量 / 急迫感增强。
- 后续跟随确认。

风险：

- 止损放在突破失败点之外。
- 如果测量目标空间已经消耗大半，避免追入。

### 交易区间反转

背景：

- 市场处在交易区间内。
- 价格到达区间上沿 / 下沿，或出现清晰的假突破。

触发：

- 区间边缘出现反转信号。
- 假突破后回到区间内部。
- 价格离开区间边缘后有跟随。

风险：

- 目标保守，通常看向区间中部或另一侧。
- 只有强突破 + 后续跟随后，才切换为趋势思维。

### 高潮 / 衰竭

背景：

- 行情已经延伸较远。
- 大 K 线和急迫感在测量目标或重要位置附近集中出现。

触发：

- 出现强反向压力，或继续推进失败。
- 反向出现后续跟随。

风险：

- 先当作反转尝试，而不是自动假设趋势已经彻底反转。
- 等确认后再期待更大级别运动。

## 交易前检查清单

```text
1. 当前背景是什么：趋势、区间、突破、回调、高潮，还是反转尝试？
2. 交易是否发生在重要结构位置？
3. 当前机会属于什么类型：突破、回调、区间反转、第二段，还是衰竭？
4. 信号 K 线在当前背景下是否足够强？
5. 如果是趋势回调，它是 H1/H2/H3 还是 L1/L2/L3？
6. 成交量是在确认急迫感，还是更像衰竭/陷阱？
7. 信号后有没有后续跟随？
8. 如果这是交易区间，我是否错误地假设突破一定会成功？
9. 到测量目标还有没有足够空间？
10. 交易逻辑失效点在哪里？
11. 多大的仓位能把单笔亏损控制在风险预算内？
12. 什么市场反馈会让我持有、减仓、平推或退出？
13. 这笔交易是否属于可重复的概率流程，而不是一次性预测？
```

## 工程化转换说明

这些概念目前还不是机械交易规则。若要转成代码或筛选器，需要先定义可检测特征：

- 市场背景：可用均线、ATR 收缩/扩张、摆动结构、K 线重叠程度来区分趋势和区间。
- 关键位置：可用前高前低、区间边界、均线、趋势线、测量目标投射来表示。
- 信号强度：可用实体占比、收盘位置、区间扩张、方向性来近似。
- 成交量急迫感：可用成交量相对滚动均量的倍数来近似。
- 后续跟随：可检测后 1 到 3 根 K 线是否继续同方向收盘、是否守住突破位、是否立即回到区间内。
- 交易区间假突破：必须先识别区间背景，再判断突破是否失败。
- 第二段逻辑：需要追踪第一段冲击、回调深度和再次推进。
- H1/H2/H3 与 L1/L2/L3：需要识别趋势方向、回调尝试、第一次恢复趋势尝试是否失败或偏弱，以及第二次入场信号。
- 风险过滤：需要计算失效点距离，并剔除盈亏比或仓位不可接受的机会。


## 与 YC-Buy Selector 的关系

YC-buy 当前重点是 13 个买点和三重滤网趋势过滤。方方土方法论可以作为更高层的过滤框架：

- 接受买点前，先做背景分类。
- 交易区间里的突破信号，如果没有后续跟随，应降低评分。
- 每个候选都必须给出明确失效点和风险预算。
- 加入测量目标空间检查，避免后段追入。
- 强突破或强反转后，跟踪第二段机会。
- 成交量只作为确认，不作为孤立信号。

潜在评分项：

```text
+ 背景与买点方向一致
+ 信号出现在关键结构位置
+ 强收盘和相对成交量确认急迫感
+ 后续跟随确认突破/反转
+ 测量目标空间支持合理盈亏比
-- 区间背景中的弱突破
-- 信号后无后续跟随
-- 止损距离过大，超过风险预算
-- 接近测量目标后的追入
```
