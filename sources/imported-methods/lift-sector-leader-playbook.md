# LIFT 板块龙头识别 Playbook

这是一份每日执行清单，用于盘前、盘中和盘后识别板块龙头。

## 1. 盘前准备

```text

昨日涨停最多板块：
昨日成交额放大板块：
今日潜在催化：
需要重点观察的板块：
```

## 2. 板块有效性检查

```text
板块名称：
板块是否强于大盘：
板块指数是否突破关键位：
板块是否放量：
板块内涨停数量：
板块内涨幅大于 5% 个股数量：
板块内创新高个股数量：
是否有明确催化：
是否有 2-3 天跟随：
```

评分：

```text
板块有效性：__/20
```

如果板块有效性低于 12 分，不进入龙头筛选。

## 3. 候选股池

从板块内列出 5 到 10 只候选：

```text
代码：
名称：
涨幅：
成交额：
换手：
启动日期：
是否涨停：
是否创新高：
是否强于板块：
是否有辨识度：
```

候选来源：

- 最早启动。
- 今日最强。
- 成交额最大。
- 最先创新高。
- 回调最抗跌。
- 板块修复时最先表态。

## 4. LIFT 打分

```text
标的：

板块有效性：__/20
Lead 领先性：__/15
Impact 带动性：__/15
Flow 资金强度：__/15
Trend 趋势结构：__/15
逻辑正宗度：__/10
风险位置：__/10
总分：__/100
```

解释：

```text
80-100：核心龙头候选。
65-79：强势跟随股。
50-64：普通板块股。
0-49：剔除。
```

## 5. 龙头类型标注

每只高分候选必须标注类型：

```text
总龙头 / 板块龙头 / 趋势龙头 / 弹性龙头 / 容量龙头 / 补涨龙头 / 反包龙头 / 防御龙头
```

避免只写“龙头”。必须说明它是哪一种龙头。

## 7. 与 BSA 联动

LIFT 只决定候选地位，不能直接决定买入。

交易前必须再做 BSA：

```text
B 背景：
S 结构：
A 行动：
入场条件：
止损位置：
第一目标：
无跟随退出：
```

规则：

```text
LIFT 高分 + BSA 好位置 = 可重点计划。
LIFT 高分 + BSA 追高位 = 等回调或放弃。
LIFT 低分 + BSA 好位置 = 只当普通交易。
```

## 8. 龙头地位复盘

```text
今日原龙头：
今日新候选：
地位是否转移：
转移证据：
板块是否继续有效：
板块是否退潮：
明日观察核心：
```

地位转移证据：

- 新候选先于原龙头修复。
- 新候选带动板块扩散。
- 原龙头高位滞涨或跌破关键位。
- 资金从原分支切换到新分支。
- 补涨变成情绪核心。

## 9. 禁止清单

- 不把当天涨幅第一直接当龙头。
- 不在板块无效时强行找龙头。
- 不把补涨股误判为主龙头。
- 不忽略成交额和流动性。
- 不忽略地位转移。
- 不因为龙头地位高就无脑追高。
- 不把消息正宗度当作唯一标准。
- 不用单日表现否定完整趋势。
