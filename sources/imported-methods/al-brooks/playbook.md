# Al Brooks Price Action Playbook

这是一份短版执行剧本，用于日常看盘、模拟交易和复盘。

## 1. 开盘前或选股后

先回答：

```text
标的：
周期：
大周期状态：
小周期状态：
Always In 方向：
关键支撑：
关键阻力：
当前价格位置：
```

市场状态只允许选一个主状态：

```text
强趋势 / 通道趋势 / 交易区间 / 突破阶段 / 反转尝试 / 不清楚
```

如果选的是“不清楚”，默认不交易。

## 2. 买卖压力阅读

观察最近 5 到 20 根K线：

```text
强多证据：
强空证据：
中性证据：
突破是否有跟随：
回调是否弱：
反转是否有二次信号：
```

判断规则：

- 连续强K线加好收盘，代表主动方强。
- 强K线没有跟随，代表信号质量下降。
- 重叠多、影线多、来回打脸，代表区间或不清楚。
- 突破后快速回到区间内，优先考虑失败突破。

## 3. 只做四类交易

### 3.1 顺势回调

适用：

- 强趋势或通道趋势。
- Always In 方向清楚。
- 回调弱。

触发：

- 上涨趋势中的 H1/H2。
- 下跌趋势中的 L1/L2。
- 回调到均线、趋势线或前突破位后出现顺势信号。

放弃：

- 回调变成强反向趋势。
- 价格已经接近通道极端位置。
- 信号K线弱且没有空间。

### 3.2 突破回调

适用：

- 关键位被强实体突破。
- 突破后有跟随。
- 回踩不深。

触发：

- 多头突破后回踩突破位，出现多头信号。
- 空头跌破后反抽跌破位，出现空头信号。

放弃：

- 突破后没有跟随。
- 回踩直接回到原区间深处。
- 回踩K线反向力量太强。

### 3.3 区间失败突破

适用：

- 市场处于交易区间。
- 价格到达区间上沿或下沿。
- 突破后无跟随并回到区间内。

触发：

- 上破失败后出现空头信号。
- 下破失败后出现多头信号。
- 第二次突破失败更优。

放弃：

- 突破出现强跟随。
- 区间边界不清楚。
- 价格在区间中间。

### 3.4 主要趋势反转

适用：

- 原趋势持续较久。
- 出现衰竭、过冲或楔形。
- 趋势线已被突破。
- 回测原极值失败。

触发：

- 下跌趋势后形成更高低点和强多头信号。
- 上涨趋势后形成更低高点和强空头信号。
- 二次信号优先于第一次信号。

放弃：

- 只有单根逆势K线。
- 原趋势还很强。
- 没有趋势线突破和回测过程。

## 4. 信号评分

每个潜在交易从 5 个维度打分，每项 0 到 2 分。

```text
市场环境：
结构位置：
K线质量：
后续跟随：
风险回报：
总分：
```

解释：

```text
8-10 分：可重点跟踪。
5-7 分：普通机会，只做轻仓或短目标。
0-4 分：放弃。
```

如果没有后续跟随，评分必须下调。

## 5. 入场前最后一句话

入场前必须能写出：

```text
我做的是：
市场状态是：
Always In 方向是：
入场信号是：
止损位置是：
第一目标是：
如果没有跟随，我会：
```

写不出来就不交易。

## 7. 禁止清单

- 不在区间中间交易。
- 不追没有跟随的突破。
- 不把第一次逆势信号当主要趋势反转。
- 不在强趋势中过早逆势。
- 不做止损无法定义的交易。
- 不因为一个术语出现就交易。
- 不在市场状态不清楚时强行解释。
