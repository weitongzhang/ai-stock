# A股选股系统使用指南

## 📖 快速上手

### 第一步：安装依赖

```bash
cd ~/a-stock-selector
pip install -r requirements.txt
```

**依赖说明：**
- `pandas`: 数据处理
- `numpy`: 数值计算
- `akshare`: A股数据获取（可选，如果不安装会使用示例数据）

### 第二步：快速测试

先运行测试脚本，确保系统正常工作：

```bash
python test.py
```

这会使用示例数据分析3只股票，生成 `test_results.csv` 文件。

### 第三步：正式运行

```bash
python main.py
```

根据提示选择：
- **选项1**：快速测试模式（10只股票）
- **选项2**：完整模式（所有A股）

## 📊 输出文件说明

### CSV文件格式

生成的CSV文件包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 股票代码 | 6位数字代码 | 600519 |
| 股票名称 | 股票名称 | 贵州茅台 |
| 符合买点数量 | 符合的买点个数 | 3 |
| 选股原因 | 详细列出所有符合的买点 | 买点3: 二次测试; 买点8: 大形态突破 |
| 最新价格 | 分析时的收盘价 | 1650.50 |
| 分析日期 | 分析日期 | 2026-03-01 |

### 文件命名

自动生成的文件名格式：`a_stock_selection_YYYYMMDD_HHMMSS.csv`

例如：`a_stock_selection_20260301_143025.csv`

## 🎯 13个买点详解

### 底部反转结构（1-4）

#### 买点1: 2B结构
**特征：**
- 完整的五浪下跌
- 大阴线跌破平台
- 第二根K线大阳线快速拉起

**适用场景：** 底部反转，机构抢筹

**风险等级：** ⭐⭐（低风险）

#### 买点2: 底部头肩右肩
**特征：**
- 形成头肩底形态
- 价格回调到50%位置

**适用场景：** 提前埋伏

**风险等级：** ⭐⭐⭐（中等风险）

#### 买点3: 二次测试
**特征：**
- 刺破前低但快速收回
- 有长下影线或反包

**适用场景：** 底部确认

**风险等级：** ⭐⭐（低风险）

#### 买点4: 强势出现
**特征：**
- 底部形态中轴以上
- 带量阳K线

**适用场景：** 吸筹后期

**风险等级：** ⭐⭐⭐（中等风险）

### 形态突破（5-8）

#### 买点5-7: 双突破
**特征：**
- 趋势线突破
- 同时突破量能密集区/关键点位/形态

**适用场景：** 突破确认

**风险等级：** ⭐⭐（低风险）

#### 买点8: 大形态突破
**特征：**
- 长期底部盘整（几个月到几年）
- 突然放量突破

**适用场景：** 大行情启动

**风险等级：** ⭐（极低风险）

### 上涨趋势中（9-13）

#### 买点9: 斐波那契回调
**特征：**
- 回调到0.382、0.5或0.618位置
- 有反转信号

**适用场景：** 趋势中的回调买入

**风险等级：** ⭐⭐（低风险）

#### 买点10: 箱体弹簧
**特征：**
- 跌破箱体底部
- 快速拉回，长下影线

**适用场景：** 震仓后买入

**风险等级：** ⭐⭐⭐（中等风险）

#### 买点11: 多周期共振
**特征：**
- 多个周期都是上涨趋势
- 多头排列

**适用场景：** 趋势加速

**风险等级：** ⭐（极低风险）

#### 买点12: 支撑压力互换
**特征：**
- 前期压力位变成支撑
- 回踩确认

**适用场景：** 趋势延续

**风险等级：** ⭐⭐（低风险）

#### 买点13: 趋势急跌
**特征：**
- 上涨趋势中第一次快速暴跌
- 跌到50%位置

**适用场景：** 抄底反弹

**风险等级：** ⭐⭐⭐⭐（高风险）

## 💡 使用技巧

### 1. 结合量比筛选

在选出的股票中，优先关注量比 > 1.5 的股票：

```python
# 在main.py中添加量比过滤
if data['volume'].iloc[-1] / data['volume'].mean() > 1.5:
    # 这是有资金关注的股票
```

### 2. 关注多买点共振

符合买点数量越多，成功概率越高：
- **3个以上买点**：重点关注
- **2个买点**：可以考虑
- **1个买点**：需要结合其他因素

### 3. 配合大盘环境

- **大盘上涨**：所有买点都可以考虑
- **大盘震荡**：优先考虑买点1-4（底部反转）
- **大盘下跌**：空仓等待

### 4. 设置止损

每个买点都应该设置止损：
- **买点1-4**：止损设在最近低点下方3%
- **买点5-8**：止损设在突破点下方5%
- **买点9-13**：止损设在支撑位下方3%

### 5. 分批建仓

不要一次性买入：
- 第一次买入30%
- 确认后加仓30%
- 突破后再加仓40%

## 🔧 自定义分析

### 只分析特定股票

修改 `main.py`：

```python
# 只分析这些股票
my_stocks = ['600519', '000333', '000858']
selector.select_stocks(stock_codes=my_stocks)
```

### 修改日期范围

```python
# 分析最近3个月的数据
from datetime import datetime, timedelta
end_date = datetime.now().strftime('%Y-%m-%d')
start_date = (datetime.now() - timedelta(days=90)).strftime('%Y-%m-%d')

selector.select_stocks(start_date=start_date, end_date=end_date)
```

### 调整买点灵敏度

在 `strategies/buy_points.py` 中可以调整各个买点的参数：

```python
# 例如：调整2B结构的涨跌幅阈值
k1_drop < -0.03  # 改为 -0.05 会更严格
k2_rise > 0.03   # 改为 0.05 会更严格
```

## 📈 实战流程

### 每日选股流程

1. **早盘前（9:00-9:25）**
   ```bash
   python main.py
   ```
   选择快速模式，分析重点关注的股票

2. **查看结果**
   - 打开生成的CSV文件
   - 按"符合买点数量"排序
   - 重点关注3个以上买点的股票

3. **人工复核**
   - 打开交易软件查看K线图
   - 确认买点是否真实有效
   - 查看量比、资金流向

4. **制定交易计划**
   - 确定买入价格
   - 设置止损位
   - 计算仓位

5. **执行交易**
   - 开盘后观察
   - 符合条件时买入
   - 严格执行止损

### 周末复盘流程

1. **完整扫描**
   ```bash
   python main.py
   ```
   选择完整模式，分析所有A股

2. **建立股票池**
   - 将符合条件的股票加入自选
   - 分类管理（底部反转、突破、趋势延续）

3. **深度研究**
   - 研究基本面
   - 查看行业地位
   - 分析财务数据

4. **制定下周计划**
   - 确定重点关注股票
   - 设定买入条件
   - 准备资金

## ⚠️ 注意事项

### 1. 技术分析的局限性

- 技术分析不是万能的
- 需要结合基本面
- 需要考虑市场环境

### 2. 避免过度交易

- 不是每个买点都要买
- 选择最有把握的机会
- 控制交易频率

### 3. 资金管理

- 单只股票不超过总资金的20%
- 同一板块不超过总资金的40%
- 保留30%现金应对机会

### 4. 心态管理

- 接受一定概率的失败
- 严格执行止损
- 不要追涨杀跌

## 🐛 常见问题

### Q1: 提示"未安装akshare"怎么办？

**A:** 安装akshare：
```bash
pip install akshare
```

如果安装失败，系统会自动使用示例数据。

### Q2: 分析速度太慢怎么办？

**A:**
1. 先用快速模式测试
2. 只分析重点股票
3. 使用多进程（需要修改代码）

### Q3: 如何提高准确率？

**A:**
1. 结合量比筛选（量比>1.5）
2. 关注多买点共振（3个以上）
3. 配合资金流向
4. 人工复核K线图

### Q4: CSV文件乱码怎么办？

**A:**
- 使用Excel打开时选择UTF-8编码
- 或者用记事本打开，另存为ANSI编码

### Q5: 能否用于港股/美股？

**A:**
- 理论上可以，但需要修改数据获取模块
- 13个买点策略是通用的
