## 📊 三重滤网核心参数

### 第一层滤网（周线）

| 参数 | 值 | 说明 |
|------|-----|------|
| MACD | 12,26,9 | 标准参数 |
| 趋势判断 | 连续3周 | MACD柱状图方向 |
| 快均线 | 5周 | 短期趋势 |
| 慢均线 | 20周 | 长期趋势 |
| 价格容差 | ±3% | 判断"附近"的标准 |

**趋势强度分级**:
- **强向上**: MACD上升 + 均线多头 + 价格在20周线上方
- **弱向上**: MACD上升 + 均线多头 + 价格在20周线附近
- **震荡**: MACD走平 + 均线粘合
- **向下**: MACD下降 或 均线空头

### 第二层滤网（日线）

| RSI区间 | 状态 | 评分 | 操作建议 |
|---------|------|------|---------|
| 0-30 | 超卖 | 10分 | 最佳买入区 |
| 30-40 | 偏弱 | 7分 | 可考虑 |
| 40-50 | 中性偏弱 | 4分 | 谨慎 |
| 50-60 | 中性 | 2分 | 不追高 |
| 60-70 | 偏强 | 0分 | 观望 |
| 70-100 | 超买 | 0分 | 不买入 |

**反转信号**:
- 长下影线：下影 > 实体2倍
- 锤子线：下影 > 实体2倍 且 上影 < 实体30%
- 阳包阴：阳线完全包住前一根阴线
- 早晨之星：三K线组合

**位置质量**:
- **极佳**: RSI<30 + 反转信号 + 缩量止跌 (10分)
- **良好**: RSI<40 + 反转信号 + 有支撑 (7分)
- **一般**: RSI<50 + 有支撑 (4分)
- **不佳**: RSI>50 或 无支撑 (0分)

### 第三层滤网（60分钟）

**突破类型**:

1. **突破前高**（追涨型）
   - 突破前3根K线高点
   - 成交量 > 前5根均量1.5倍
   - 阳线实体 > 1%

2. **反弹确认**（抄底型）
   - 从日内低点反弹 > 2%
   - 有反转K线
   - 成交量温和放大（1.2倍）

3. **均线突破**
   - 5周期均线上穿10周期均线
   - MACD金叉

**确认条件**:
- 突破后3根K线不跌破
- 成交量维持在均量以上
- 收盘价站在突破位上方

## 🎯 组合规则和评分

### 信号分级

| 信号类型 | 评分 | 仓位 | 止损 | 条件 |
|---------|------|------|------|------|
| 买入信号 | 10分 | 30-50% | 3% | 三层全满足 |
| 等待信号 | 6分 | 10-20% | 2% | 前两层满足 |
| 可考虑 | 4分 | 10-20% | 2% | 趋势+突破 |
| 观望 | 0分 | 0% | - | 不满足 |

### 与13个买点的配合

**最强组合（A级）**:
- 三重滤网: 买入信号（10分）
- 13个买点: 总分≥15 且 有买点1或8
- 总分: ≥25分
- 评价: 强烈推荐

**次强组合（B级）**:
- 三重滤网: 买入信号（10分）
- 13个买点: 总分≥12 且 符合≥3个买点
- 总分: 22-24分
- 评价: 推荐

**一般组合（C级）**:
- 三重滤网: 等待信号（6分）
- 13个买点: 有买点8（大形态突破）
- 总分: 15-21分
- 评价: 可考虑

**放弃（D级）**:
- 总分: <15分
- 评价: 放弃

### 总评分计算

```
总分 = 三重滤网评分 + 13个买点评分

等级划分:
A级（强烈推荐）: ≥20分
B级（推荐）: 15-19分
C级（可考虑）: 10-14分
D级（放弃）: <10分
```

## 🔧 市场环境参数调整

### 牛市参数

**调整内容**:
- 第一层: 接受weak_up（弱向上）
- 第二层: RSI超卖放宽到35，偏弱放宽到45
- 第三层: 突破确认简化到2根K线

**适用场景**: 大盘持续上涨，个股普遍强势

### 熊市参数

**调整内容**:
- 第一层: 只做up（强向上），不做weak_up
- 第二层: RSI超卖收紧到25，必须RSI回升
- 第三层: 需要3根K线确认，成交量>2倍均量

**适用场景**: 大盘下跌，个股普遍弱势

### 震荡市参数

**调整内容**:
- 第一层: 接受sideways（震荡）中的向上段
- 第二层: RSI<30买入，RSI>70卖出
- 第三层: 降低预期，小波段操作

**适用场景**: 大盘横盘，个股分化
