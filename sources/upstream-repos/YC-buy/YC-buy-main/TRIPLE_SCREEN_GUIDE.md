# 三重滤网系统使用指南

## 📖 理论基础

三重滤网系统由亚历山大·埃尔德博士提出，记录于《走进我的交易室》一书中。

### 核心理念

**顺大势，逆小势，等突破**

- **顺大势**：只在长周期趋势方向交易
- **逆小势**：在中周期的回调位置入场
- **等突破**：等待短周期的突破确认

## 🎯 三层滤网详解

### 第一层滤网：长周期趋势

**目的**：判断大趋势方向

**指标**：MACD（周线级别）

**规则**：
- MACD斜率向上 → 做多
- MACD斜率向下 → 做空（A股不常用）
- MACD横盘 → 观望

**意义**：
- 确保交易方向与大趋势一致
- 避免逆势交易
- 提高胜率

### 第二层滤网：中周期震荡

**目的**：寻找最佳入场位置

**指标**：RSI（日线级别）

**规则**：
- 长周期向上 + RSI < 30（超卖）→ 准备做多
- 长周期向下 + RSI > 70（超买）→ 准备做空

**意义**：
- 在回调时买入，而不是追高
- 获得更好的盈亏比
- 降低买入成本

### 第三层滤网：短周期突破

**目的**：确认入场时机

**方法**：看盘中价格突破

**规则**：
- 向上突破前高 → 买入
- 向下跌破前低 → 卖出

**意义**：
- 等待市场确认
- 避免过早入场
- 提高成功率

## 📊 信号类型说明

### 买入信号（最强）

**条件**：三层滤网全部满足
- ✓ 长周期趋势向上
- ✓ 中周期超卖（RSI < 30）
- ✓ 短周期向上突破

**操作**：可以买入

**风险等级**：⭐（低风险）

### 等待突破（次强）

**条件**：前两层满足，等待第三层
- ✓ 长周期趋势向上
- ✓ 中周期超卖
- ✗ 短周期未突破

**操作**：密切关注，等待突破

**风险等级**：⭐⭐（低风险）

### 可考虑买入（一般）

**条件**：趋势和突破满足，但位置不佳
- ✓ 长周期趋势向上
- ✗ 中周期不在超卖区
- ✓ 短周期向上突破

**操作**：可以考虑，但盈亏比不是最佳

**风险等级**：⭐⭐⭐（中等风险）

## 💡 实战技巧

### 1. 周期选择

**中线交易**（推荐）：
- 长周期：周线
- 中周期：日线
- 短周期：日内分时

**短线交易**：
- 长周期：日线
- 中周期：60分钟
- 短周期：15分钟

**长线交易**：
- 长周期：月线
- 中周期：周线
- 短周期：日线

### 2. 止损设置

**买入信号**：
- 止损设在当日最低点下方2-3%

**等待突破**：
- 如果突破失败，立即止损

**可考虑买入**：
- 止损设在前期支撑位下方

### 3. 仓位管理

**买入信号**：
- 可以用30-50%仓位

**等待突破**：
- 先用10-20%试探
- 突破后加仓

**可考虑买入**：
- 只用10-20%仓位

### 4. 持仓管理

**平仓条件**：
- 长周期趋势反转（MACD死叉）
- 中周期进入超买区（RSI > 70）
- 短周期跌破关键支撑

**加仓条件**：
- 买入后盈利10%以上
- 再次出现"等待突破"信号
- 可以加仓20-30%

## 📈 使用流程

### 每日选股流程

1. **早盘前（9:00-9:25）**
   ```bash
   python triple_screen_main.py
   ```
   选择快速模式

2. **查看结果**
   - 优先关注"买入信号"
   - 其次关注"等待突破"
   - "可考虑买入"需谨慎

3. **人工复核**
   - 打开K线图确认
   - 查看成交量
   - 确认趋势

4. **制定计划**
   - 确定买入价格
   - 设置止损位
   - 计算仓位

5. **执行交易**
   - 开盘后观察
   - 符合条件时买入
   - 严格执行止损

### 周末复盘流程

1. **完整扫描**
   ```bash
   python triple_screen_main.py
   ```
   选择完整模式

2. **建立股票池**
   - 将符合条件的股票加入自选
   - 按信号类型分类

3. **深度研究**
   - 研究基本面
   - 查看行业地位
   - 分析财务数据

4. **制定下周计划**
   - 确定重点关注股票
   - 设定买入条件

## ⚠️ 常见错误

### 错误1：忽略第一层滤网

**表现**：在下跌趋势中抄底

**后果**：逆势交易，胜率低

**正确做法**：只在上涨趋势中做多

### 错误2：不等第三层滤网

**表现**：看到超卖就买入

**后果**：过早入场，可能继续下跌

**正确做法**：等待突破确认

### 错误3：追高买入

**表现**：在RSI > 70时追涨

**后果**：买在高位，盈亏比差

**正确做法**：等待回调到超卖区

### 错误4：不设止损

**表现**：买入后不设止损

**后果**：小亏变大亏

**正确做法**：买入前就设好止损

## 📊 与13个买点系统的配合

### 组合使用策略

1. **用三重滤网筛选大方向**
   - 找出长周期趋势向上的股票
   - 确定大盘环境

2. **用13个买点找具体时机**
   - 在三重滤网筛选出的股票中
   - 寻找具体的买入点

3. **双重验证**
   - 三重滤网：买入信号
   - 13个买点：符合2个以上买点
   - 这样的股票成功率最高

### 实战流程

```bash
# 步骤1：用三重滤网筛选
python triple_screen_main.py

# 步骤2：将结果中的股票代码提取出来
# 例如：000333, 600519, 000001

# 步骤3：用13个买点系统分析这些股票
# 修改main.py，只分析这些股票
python main.py

# 步骤4：选择同时满足两个系统的股票
```

## 🔧 参数调整

### RSI阈值

默认：30/70

可以调整为：
- 更严格：25/75
- 更宽松：35/65

在 `strategies/triple_screen.py` 中修改：

```python
if rsi_current < 30:  # 改为25或35
    position = 'oversold'
```

### MACD参数

默认：12/26/9

可以调整为：
- 更灵敏：6/13/5
- 更平滑：19/39/9

### 突破阈值

默认：0.5%

可以调整为：
- 更严格：1%
- 更宽松：0.3%
