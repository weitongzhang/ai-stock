# A股选股系统 - 双系统版

一个基于道氏理论、威科夫理论和波浪理论的A股技术分析选股系统。

## 🎯 两大选股系统

### 系统1: 13个买点策略
基于经典技术分析理论，识别从底部到顶部的各个买入时机。

### 系统2: 三重滤网系统
基于亚历山大·埃尔德博士的理论，核心是"顺大势，逆小势，等突破"。

## 📊 13个买点策略

### 底部反转结构的买点（1-4）

1. **2B结构（云聪版）**
   - 完整的五浪下跌
   - 大阴线跌破平台后大阳线快速拉起
   - 概率高达90%

2. **底部头肩的右肩（50%位置）**
   - 价格回调到前一个下跌波段的50%位置
   - 提前埋伏买入

3. **二次测试**
   - 刺破前低但很快收回
   - 伴有明确K线信号（长下影、反包）

4. **强势出现（威科夫概念）**
   - 底部形态中轴以上
   - 出现带量阳K线

### 形态突破和中继的买点（5-8）

5. **趋势线 + 量能密集区的突破**
6. **趋势线 + 关键点位的突破**
7. **趋势线 + 形态的突破**
8. **大形态突破**
   - 长期底部盘整后的突破
   - 茶杯带柄、老鸭头模型

### 上涨趋势中的买点（9-13）

9. **波段的回踩（斐波那契回调）**
   - 回调到0.382、0.5或0.618位置

10. **箱体/形态内的弹簧**
    - 跌破箱体底部但快速拉回

11. **多周期共振（趋势加速）**
    - 多个周期都呈现上涨趋势

12. **支撑压力互换**
    - 前期压力转换为支撑

13. **趋势急跌（首次急跌到50%）**
    - 上涨趋势中第一次快速暴跌

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install pandas numpy akshare
```

### 2. 选择使用哪个系统

#### 系统1: 13个买点策略（推荐日内选股）

```bash
cd ~/a-stock-selector
python test.py              # 快速测试
python main.py              # 正式运行
```

**适用场景**：
- 寻找底部反转机会
- 识别突破信号
- 趋势中的回调买点

#### 系统2: 三重滤网系统（推荐中长线）

```bash
cd ~/a-stock-selector
python test_triple_screen.py      # 快速测试
python triple_screen_main.py      # 正式运行
```

**适用场景**：
- 顺大势，逆小势
- 等待最佳入场时机
- 中长线持仓

### 3. 查看结果

两个系统都会自动生成CSV文件：

**13个买点系统输出**：
- 股票代码、股票名称
- 符合买点数量
- 选股原因（详细列出所有符合的买点）
- 最新价格、分析日期

**三重滤网系统输出**：
- 股票代码、股票名称
- 信号类型（买入信号/等待突破/可考虑买入）
- 长周期趋势、中周期位置、短周期突破
- RSI、最新价格、分析说明

## 🔍 两个系统的区别

### 13个买点系统

**特点**：
- ✅ 覆盖完整的交易周期（底部→突破→趋势）
- ✅ 多个买点可以组合验证
- ✅ 适合短线和波段交易
- ✅ 信号明确，容易执行

**适合人群**：
- 喜欢短线交易的投资者
- 追求高频交易机会
- 善于把握技术形态

### 三重滤网系统

**特点**：
- ✅ 多周期验证，降低风险
- ✅ 顺大势，逆小势，等突破
- ✅ 适合中长线持仓
- ✅ 信号严格，成功率高

**适合人群**：
- 喜欢中长线交易的投资者
- 追求稳健收益
- 善于等待最佳时机

## 💡 如何选择

### 建议组合使用

1. **用三重滤网选方向**
   - 确定大趋势
   - 找到符合"顺大势"的股票

2. **用13个买点找时机**
   - 在三重滤网筛选出的股票中
   - 寻找具体的买入点

3. **双重验证**
   - 如果一只股票同时满足两个系统
   - 成功概率会大大提高

## 📚 理论基础

本系统基于以下经典技术分析理论：

1. **道氏理论**：趋势、支撑压力、形态
2. **威科夫理论**：量价结构、吸筹派发
3. **波浪理论**：斐波那契回调、波浪结构
